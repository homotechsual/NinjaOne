# Get-NinjaOneUserRoles

## SYNOPSIS
Gets user roles.

## SYNTAX

```
Get-NinjaOneUserRoles [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Retrieves all user roles via the NinjaOne v2 API.

## EXAMPLES

### EXAMPLE 1
```
Get-NinjaOneUserRoles
```

Gets all user roles.
