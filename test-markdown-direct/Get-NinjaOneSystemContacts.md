# Get-NinjaOneSystemContacts

## SYNOPSIS
Gets system contacts from the NinjaOne API.

## SYNTAX

```
Get-NinjaOneSystemContacts [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Retrieves system contacts from the NinjaOne v2 API.

## EXAMPLES

### EXAMPLE 1
```
Get-NinjaOneSystemContacts
```

Gets all system contacts.
