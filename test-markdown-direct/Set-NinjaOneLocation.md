# Set-NinjaOneLocation

## SYNOPSIS
Sets location information, like name, address, description etc.

## SYNTAX

```
Set-NinjaOneLocation [-organisationId] <Int32> [-locationId] <Int32> [-locationInformation] <Object>
 [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Sets location information using the NinjaOne v2 API.
