# Get-NinjaOneGroups

## SYNOPSIS
Gets groups from the NinjaOne API.

## SYNTAX

```
Get-NinjaOneGroups [[-languageTag] <String>] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Retrieves groups from the NinjaOne v2 API.

## EXAMPLES

### EXAMPLE 1
```
Get-NinjaOneGroups
```

Gets all groups.
